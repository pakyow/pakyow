There you have it&mdash;you were able to build a complete app with just a few lines of HTML. Pakyow uses a built-in tool called reflection to understand the semantic intent of your view templates and then bootstraps a working backend for your interface. From here, you can extend the reflection to add any custom features that you might need.

We covered a lot of ground in this example, introducing several concepts including view composition, data bindings, and reflection. Use the links below to learn more about the parts of the framework that we covered here:
